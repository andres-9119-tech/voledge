# VolEdge v1.0

Aplicación PWA estática para analizar estrategias Mover–Moocher con long straddles.

## Publicar gratis con GitHub Pages
1. Sube todos los archivos de esta carpeta a la raíz del repositorio `voledge`.
2. En GitHub abre: Settings → Pages.
3. En Build and deployment selecciona `Deploy from a branch`.
4. Branch: `main`; carpeta: `/ (root)`.
5. Guarda y espera a que GitHub publique la URL.

## Instalar en móvil
- iPhone: abre la URL en Safari → Compartir → Añadir a pantalla de inicio.
- Android: abre la URL en Chrome → Instalar aplicación.

## Datos
Los registros se guardan en el navegador mediante localStorage.
Usa Exportar / Importar para hacer copias de seguridad.

## Incluye
- Dashboard y ranking
- Calendario de eventos
- Diario de operaciones
- Score automático
- IV y VI/Vol. Hist.
- Capturas de IBKR
- PWA instalable
- Modo offline básico
